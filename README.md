# Fake_New_Detection
A simple model to detect Real vs Fake News with Python using 
TfidfVectorizer and PassiveAggressiveClassifier

## DataSource
The dataset is from DataFlair. This dataset has a shape of 7796Ã—4. The first column identifies the news, the second and third are the title and text, and the fourth column has labels denoting whether the news is REAL or FAKE.
